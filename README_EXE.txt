ΠΩΣ ΝΑ ΤΟ ΚΑΝΕΙΣ EXE

1. Κάνε Extract All το ZIP.
2. Άνοιξε με διπλό κλικ το BUILD_EXE.bat.
3. Περίμενε να τελειώσει.
4. Θα δημιουργηθεί φάκελος dist.
5. Μέσα θα βρεις:
   Love975MusicResearch.exe

Από εκεί και μετά τρέχεις μόνο αυτό το exe.

Όταν ανοίγει:
- ξεκινάει τον server
- ανοίγει μόνο του το admin
- admin: http://127.0.0.1:5000/admin
- έρευνα: http://127.0.0.1:5000

Για public link σε ακροατές, ανοίγεις και ngrok:
ngrok http 5000
